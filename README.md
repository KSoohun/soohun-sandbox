## Alert Component Demos

This repository contains static HTML demos of the Alert component
for the Ontada Design System, generated using different tools for comparison.

### Demos

- **Claude-generated Alert demo**  
  https://ksoohun.github.io/ontada-ds/claude-alert.html

- **Figma Make–generated Alert demo**  
  https://ksoohun.github.io/ontada-ds/figma-make-alert.html

### Notes

- These demos are exploratory and intended for design and accessibility review.
- No production frameworks are used.
- Focus is on structure, tokens, variants, and WCAG considerations.# ontada-ds
