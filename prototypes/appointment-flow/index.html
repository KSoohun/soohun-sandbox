📚 Available Components

  • Button (React) - Flexible button with variants, sizes, and loading states
  • Alert (Web Component) - Native alert component with multiple variants
  • Design Tokens - Complete token system for colors, spacing, typography

🎨 Design Tokens

Use consistent design tokens throughout your components:

import { colors, spacing, typography } from './tokens'; const primaryColor = colors.primary[500]; const mediumSpacing = spacing[4]; const baseFont = typography.fontSize.base;
